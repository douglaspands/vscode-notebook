# vscode-notebook
Projeto de Jupyter Notebook configurado no VSCode.

## Como usar
### Requisitos
- Python >=3.9
- Poetry

### Instalar dependencias
```sh
poetry install
```

## Notebooks
- [Download de manga do MangaLivre](./vscode_notebook/mangalivre.ipynb)
- [Download de video do Youtube](./vscode_notebook/youtube.ipynb)
